# Commons-Box
Common utilities needed for Java and Minecraft.

## Maven repo
Add repository
```xml
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
```
Then add the dependency
```xml
    <dependencies>
            <dependency>
            <groupId>com.github.Despical</groupId>
            <artifactId>Commons-Box</artifactId>
            <version>1.0.5</version>
            <scope>compile</scope>
        </dependency>
    </dependencies>
```
